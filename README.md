# PencilSoul Studio — Custom Pencil Sketch Portraits

A single‑page website for **PencilSoul Studio**, offering high‑quality custom pencil sketch portraits.  
The site is entirely self‑contained in a single HTML file and is designed to be deployed instantly via GitHub Pages.

---

## 🎨 What It Is

A minimalist landing page built to showcase the studio’s style. It features:

- Elegant typography and a warm, classy colour palette
- Inline sample sketch images (embedded as base‑64 data)
- Responsive layout (works on desktop and mobile)
